A comma separated list of triggers (TriggerCauses) to blacklist. It is necessary to specify the full class names including the package for the blacklisting to work.
When specified, tasks triggered by the blacklisted triggers won't be scheduled by Jenkins.

The blacklist defaults to hudson.triggers.TimerTrigger$TimerTriggerCause and hudson.triggers.SCMTrigger$SCMTriggerCause if left blank.
